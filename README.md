# LowesCampusHackathon
Lowes Campus Hackathon problem statement 3 submission

Instructions:

1. Change the working directory in the code acccording to the server.
2. Change the name of MySQL database service provider. Also change the MySQL user and password
3. Create table called shop (shopid,username,email,password)
4. Create table items (shopid,icode,iname,x,y,level) to store the items in each shop.
5. The input file to be uploaded must have a number denoting the level of the floor.
      The shop can have multiple floors . Hence the name of file can be level1,input1,input2 and so on.
      The file will be stored on the server as (shopid_level_filename) ex-1_1_input1.csv
